Jurusan yang membekali peserta didik dengan pengetahuan, keterampilan, dan sikap unggul agar siap menjadi tenaga kompeten di bidang rekayasa perangkat lunak.

Membangun Masa Depan Melalui Kode

Rekayasa Perangkat Lunak (RPL) / PPLG merupakan program keahlian yang mempelajari proses pembuatan perangkat lunak secara lengkap, mulai dari analisis kebutuhan pengguna, perancangan sistem, pemrograman, pengujian, hingga pemeliharaan aplikasi setelah digunakan. Program ini bertujuan membekali siswa dengan pengetahuan, keterampilan, dan sikap profesional agar mampu bersaing di dunia industri teknologi.

Web Dev

Menguasai Frontend (React) & Backend (NodeJS/PHP).

Game Dev

Pembuatan aset 2D/3D dan mekanik gim menggunakan Unity.
